## Retning

Bruk property "orientation" for å bestemme retning på radio-knappene

<div class="container">
  <span class="elem">
    <Canvas of={NveRadioStories.Horizontal} />
  </span>
  <span class="elem">
    <Canvas of={NveRadioStories.Vertical} />
  </span>
</div>

## Disabled

Bruk property "disabled" på nve-radio-komponenten for å disable radio-knappene

<div class="container">
  <span class="elem">
    <Canvas of={NveRadioStories.Enabled} />
  </span>
  <span class="elem">
    <Canvas of={NveRadioStories.Disabled} />
  </span>
</div>

## Obligatorisk

Bruk property "required" for å markere knapper som obligatoriske

<div class="container">
  <span class="elem">
    <Canvas of={NveRadioStories.Required} />
  </span>
  <span class="elem">
    <Canvas of={NveRadioStories.Optional} />
  </span>
</div>

## Størrelse

Bruk property "size" for å endre størrelse

<div class="container">
  <span class="elem">
    <Canvas of={NveRadioStories.Small} />
  </span>
  <span class="elem">
    <Canvas of={NveRadioStories.Medium} />
  </span>
  <span class="elem">
    <Canvas of={NveRadioStories.Large} />
  </span>
</div>

## Checked

Sett property "value" for radio-button-group lik value-verdien for radio-button for å huke av for aktuell knapp

<div class="container">
  <span class="elem">
    <Canvas of={NveRadioStories.Valueone} />
  </span>
  <span class="elem">
    <Canvas of={NveRadioStories.Valuetwo} />
  </span>
  <span class="elem">
    <Canvas of={NveRadioStories.Valuethree} />
  </span>
</div>
