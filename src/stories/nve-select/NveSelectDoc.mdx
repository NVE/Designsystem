## Størrelse

  <span class="elem">
    <Canvas of={NveSelectStories.Small} />
  </span>

## Disabled

Bruk property "disabled" for å disable komponenten. Bruk property "disabled" på nve-option-komponenten for å disable valg (se option 7)

<div class="container">
  <span class="elem">
    <Canvas of={NveSelectStories.Enabled} />
  </span>
  <span class="elem">
    <Canvas of={NveSelectStories.Disabled} />
  </span>
</div>

## Filled

Bruk property "filled" for å fylle komponenten med bakgrunnsfarge

<div class="container">
  <span class="elem">
    <Canvas of={NveSelectStories.Unfilled} />
  </span>
  <span class="elem">
    <Canvas of={NveSelectStories.Filled} />
  </span>
</div>
